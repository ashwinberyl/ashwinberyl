### The short version

Eight years in, across Aptiv, HCLTech, and Cognizant, I've learned that good platform engineering is invisible. Nobody notices the pipeline that doesn't break, the deploy that doesn't page anyone at 2am, or the security scan that catches the problem before it ships. That's the job. I build the boring, reliable layer so 150+ developers can focus on their actual work instead of fighting infrastructure.

**What I actually do:** take fragile, manual, or unscalable systems — cloud spend, CI/CD, access control, observability — and turn them into something that runs itself.

### How I work

```text
1. Measure before you touch it.       → no optimization without a baseline
```
